# 🤖 DexTerAi - Chat dengan Honcho AI

Aplikasi web chat yang terintegrasi dengan Honcho AI (Claude Sonnet 4) dan di-deploy di Vercel.

## 🚀 Fitur

- ✨ UI modern dan responsive
- 💬 Multi-turn conversation
- 🎨 Dark mode dengan gradient
- ⚡ Fast response dengan Vercel serverless
- 🔄 Reset chat functionality
- 📱 Mobile-friendly

## 📋 Prasyarat

- Node.js 18+ (untuk development lokal)
- Akun Vercel (untuk deploy)
- Vercel CLI (opsional)
